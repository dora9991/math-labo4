📝 中1 小テスト生成ツール

問題DBから5分テストを自動編成 → Googleフォーム用スクリプト(.gs)を出力

① 出題条件

★1 基礎 ★2 標準 ★3 応用

② 苦手を優先する（任意）

アプリの「💾 保存する」で出てくるバックアップJSONを読み込むと、 習熟度の低いスキル・間違えた問題を優先して出題します。
（読み込まない場合は難易度の高い問題を優先します）
未読み込み（難易度ベースで優先します）

③ 使い方

  1. 条件を決めて「抽選」
  2. 中央のプレビューで内容を確認（差し替え・再抽選可）
  3. 「Apps Scriptを生成」をクリック
  4. 表示された .gs をコピー
  5. script.google.com で新規プロジェクト → 貼り付け → 実行
  6. 実行ログのURLを開くとフォーム完成（クイズ形式・自動採点つき）
